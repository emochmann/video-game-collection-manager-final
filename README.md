Video Game Collection Manager – Setup Instructions

1. Download and extract the project zip
2. Open the project in IntelliJ IDEA
3. Ensure Java is installed on your system
4. Make sure the sqlite-jdbc dependency is included in the project
5. Run Main.java OR run the provided JAR file

Running the JAR:
1. Open Terminal in the folder containing the JAR file
2. Run: java -jar VideoGameCollectionManager.jar
3. When prompted, select the provided games.db file

Using the Application:
* The user must select a SQLite database file (games.db)
* The application will load all records from the database
* Users can add, update, delete, and search for games
*  All changes are saved directly to the database
